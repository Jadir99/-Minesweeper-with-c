Introduction

Minesweeper is a single-player puzzle video game. The objective of the game is to clear a rectangular board containing hidden "mines" or bombs without detonating any of them, with help from clues about the number of neighboring mines in each field.

Features

  • Dynamic grid creation based on user-selected difficulty.
  • Random placement of mines within the grid.
  • Clues indicating the number of neighboring mines.
  • Flagging mechanism to mark potential mine locations.
  • Recursive algorithm for revealing adjacent empty squares.
  • Gameplay loop allowing players to interact with the grid until completion or detonation.

How to Play

  1. Choose the difficulty level: easy, medium, or hard.
  2. The game will generate a grid with mines randomly placed.
  3. Use the provided menu options to interact with the grid:
    • Add a flag to mark potential mine locations.
    • Visit a square to reveal its content.
    • Delete a flag if placed incorrectly.
  4. The game ends when either all non-mine squares are revealed (win) or a mine is detonated (lose).
